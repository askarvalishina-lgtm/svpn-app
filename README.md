# Инструкция по сборке APK для SVPN (Лучший клиент на базе Xray/v2ray)

Этот проект представляет собой стильный и функциональный клиент SVPN со своим уникальным киберпанк-стилем и динамической подгрузкой ключей/конфигов напрямую из вашего PHP-скрипта API (`https://supercretn.ps.fhgdps.com/svpn.php`).

## Как быстро собрать APK файл:

Поскольку для компиляции полноценного Android-приложения с интеграцией низкоуровневых go-библиотек Xray Core требуется локальное окружение (Android Studio и Android SDK), вы можете за 2 минуты собрать рабочий `.apk` у себя на компьютере:

### Вариант 1. Сборка через Android Studio (Самый простой и рекомендуемый)
1. Скачайте архив `svpn_android_project.zip` из этого диалога.
2. Распакуйте архив в любую удобную папку на вашем компьютере.
3. Откройте **Android Studio** и выберите **"Open"** (Открыть существующий проект).
4. Выберите распакованную папку `svpn-project`.
5. Дождитесь автоматической синхронизации Gradle-зависимостей.
6. В верхнем меню выберите: **Build -> Build Bundle(s) / APK(s) -> Build APK(s)**.
7. Спустя пару секунд появится всплывающее окно со ссылкой на готовый файл `app-debug.apk`. Его можно устанавливать на любой Android-телефон!

---

### Вариант 2. Сборка через командную строку (CLI)
Если у вас установлен Android SDK, вы можете собрать его одной командой из папки проекта:
* **Windows (PowerShell/CMD):**
  ```bash
  gradlew.bat assembleDebug
  ```
* **macOS / Linux:**
  ```bash
  chmod +x gradlew
  ./gradlew assembleDebug
  ```
Собранный файл будет лежать по пути: `app/build/outputs/apk/debug/app-debug.apk`.

## Архитектура и функционал приложения:
1. **Динамический ConfigFetcher:** При старте делает асинхронный HTTP GET запрос к вашему скрипту `https://supercretn.ps.fhgdps.com/svpn.php` и автоматически декодирует Base64 или парсит сырые протоколы (VLESS, VMess, Trojan, Shadowsocks).
2. **Умный ConfigParser:** Автоматически извлекает из URI-ссылок хосты, порты, UUID ключи и имена серверов для каждого протокола.
3. **Киберпанк UI дизайн:** Экран в футуристическом стиле (темная палитра с неоновым бирюзовым свечением `#00FFCC`, интерактивный статус-круг подключения, кастомный список быстрых профилей с индикаторами).
4. **Xray VPN-тоннель:** Встроенный `VpnService` для маршрутизации трафика и запуск сессий Xray-клиента.
